# Backend Help

## Build System
This backend uses Gradle with Wrapper only.

## Useful Commands

### Windows
- `.\gradlew.bat compileJava`
- `.\gradlew.bat test`
- `.\gradlew.bat bootRun`
- `.\gradlew.bat clean build`

### Unix/Linux/macOS
- `./gradlew compileJava`
- `./gradlew test`
- `./gradlew bootRun`
- `./gradlew clean build`
