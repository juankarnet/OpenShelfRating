# Backend

Java 21 + Spring Boot 4.1 + Gradle 8.8 with Wrapper

## Build & Run
```powershell
# Windows
.\gradlew.bat bootRun

# Unix/Linux/Mac
./gradlew bootRun
```

## Build only
```powershell
.\gradlew.bat build
```

## Commands
- `./gradlew build` - compile and package
- `./gradlew test` - run tests
- `./gradlew bootRun` - start development server
- `./gradlew clean` - clean build artifacts
